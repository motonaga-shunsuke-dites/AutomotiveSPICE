| **ID**           | UNT-001                            |

#### データ構造

```c
/* 例 */
typedef struct {
    uint8_t  status;   /* 状態 */
    uint32_t value;    /* 値   */
} ExampleData_t;
```

#### アルゴリズム

```
1. 入力値の検証
2. 状態に応じた処理分岐
   2.1. 正常時: ○○を実行
   2.2. 異常時: エラーコードを返す
3. 結果を出力
```

## 7. 動的ふるまい

### 7.1 フローチャート

```
[開始]
  │
  ▼
[入力検証] ──NG──▶ [エラー返却]
  │ OK
  ▼
[メイン処理]
  │
  ▼
[結果返却]
  │
 [終了]
```

### 7.2 状態遷移

| 現状態    | イベント   | 次状態    | アクション     |
|----------|-----------|----------|---------------|
| Idle     | start     | Running  | initProcess() |
| Running  | complete  | Idle     | cleanup()     |
| Running  | error     | Error    | logError()    |
